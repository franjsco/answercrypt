
# answercrypt

**answercrypt** is an open-source web application that uses client-side encryption to keep your secrets safe.

It is designed to be easy to use and accessible to everyone.

![answercrypt](/answercrypt.png)



## Key Features

- **Encrypt your secret with answers**  

- **Decrypt your secret with Q/A Pairs**  

- **Download QR/Payload encrypted secrets**  

- **Fetch encrypted payload from your server**
